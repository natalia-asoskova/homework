# Project: Processing Module

## Цель проекта

Данный проект содержит модуль `processing` с полезными функциями для работы со списками словарей. Основные задачи — фильтрация элементов по значению ключа `state` и сортировка по дате (`date`). Модуль предназначен для удобной обработки структурированных данных с минимальными усилиями.

## Установка

Проект не требует установки дополнительных зависимостей. Все функции реализованы на стандартном Python (версия 3.7+ рекомендуется для корректной работы функции `datetime.fromisoformat`).

- Скачайте файл `processing.py` в ваш проект.
- Импортируйте необходимые функции из модуля.
