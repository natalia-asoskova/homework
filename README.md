# Project: Processing Module

## Цель проекта

Создание новых веток в  репозитории для работы по GitFlow.
Создание нового репозитория на GitHub, который будет использоваться для хранения и совместной работы над вашим проектом.
Использование git add, git commit, git push
В модуле processing написать функцию filter_by_state, которая принимает список словарей и опционально значение для ключа 
state(по умолчанию 'EXECUTED')
В том же модуле напишите функцию sort_by_date, которая принимает список словарей и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание). Функция должна возвращать новый список, отсортированный по дате (date).
## Установка

Проект не требует установки дополнительных зависимостей.
